## 4. Фаза 1 — Критические исправления (1-2 недели)

### 4.0. СРОЧНО: Починить robots.txt (403 Forbidden)

На продакшене `https://magiacvetov12.ru/robots.txt` возвращает **403 Forbidden**. Это блокирует всех поисковых ботов от чтения файла. Нужно проверить конфигурацию nginx — возможно, статические `.txt` файлы не обслуживаются.

### 4.0.1. СРОЧНО: Добавить H1 на главную страницу

На главной странице (`frontend/src/app/page.tsx`) **отсутствует тег `<h1>`**. Это критический SEO-дефект на самой важной странице.

Рекомендуемый H1:
```html
<h1>Доставка цветов по Зеленодольску и Волжску</h1>
```

### 4.0.2. СРОЧНО: Исправить lastModified в sitemap

В `frontend/src/app/sitemap.ts` все `lastModified` установлены в `new Date()` — всегда текущая дата. Использовать реальные даты обновления или фиксированные.

### 4.1. Создать дефолтное OG-изображение

**Файл:** `frontend/public/og-image.jpg`
**Размер:** 1200x630 px
**Содержимое:** Логотип «Магия Цветов» + текст «Доставка цветов по Зеленодольску, Волжску и Казани»

**Обновить layout.tsx:**
```typescript
openGraph: {
  // ... существующие
  images: [
    {
      url: '/og-image.jpg',
      width: 1200,
      height: 630,
      alt: 'Магия Цветов — Доставка цветов по Зеленодольску и Волжску',
    },
  ],
},
```

### 4.2. Создать недостающие файлы

```bash
# public/manifest.webmanifest — PWA манифест
# public/favicon.ico — из SVG конвертировать
# public/apple-touch-icon.png — 180x180
# public/images/store.jpg — фото магазина
# public/images/logo.png — логотип для Schema.org
```

### 4.3. Добавить `<meta name="theme-color">`

В layout.tsx metadata:
```typescript
themeColor: '#DC2626', // цвет бренда
```

### 4.4. Добавить poweredByHeader: false в next.config.ts

```typescript
poweredByHeader: false,
compress: true,
```

### 4.5. Добавить trailingSlash в next.config.ts

```typescript
trailingSlash: false, // или true, но единообразно
```

### 4.6. Настроить security headers в next.config.ts

```typescript
async headers() {
  return [
    {
      source: '/(.*)',
      headers: [
        { key: 'X-Frame-Options', value: 'DENY' },
        { key: 'X-Content-Type-Options', value: 'nosniff' },
        { key: 'Referrer-Policy', value: 'strict-origin-when-cross-origin' },
      ],
    },
    {
      source: '/(.*)\\.(jpg|jpeg|png|gif|ico|svg|webp|woff2)',
      headers: [
        { key: 'Cache-Control', value: 'public, max-age=31536000, immutable' },
      ],
    },
  ];
},
```
